create-pwa

Welcome to the create-pwa template. This is a minimal pwa template using only vanilla js, with no frameworks or external dependencies. It's the perfect starting point for creating a great and fast progressive web app.

This template was created to accompany this tutorial, where all the details are explained.

What does it include?

  • Boilerplate HTML
  • Boilerplate manifest.json
  • Service worker set up for offline capability
  • iOS specific meta tags for as native-like behaviour as possible

Building on this template, your web app will get a perfect Lighthouse score (provided your host serves it with HTTPS).
